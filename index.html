toddlerMusicBox

A toddler friendly music box.

Download this project as a .zip file Download this project as a tar.gz file

My toddler is currently one and a half and loves music. My wife and I decided to search for a music playback solution for him. Our first idea was to buy a cd player radio. After my own experience of listening to mp3's for many years I did not like to buy any cd's. For a short test if let my toddler play with my ipod but he was not able to select his favorite music. Inspired by some other projects I came up with a the idea of a custom made music box for him.

Parts list

  • my old computer speakers
  • Raspberry Pi Model B+
  • EXPLORE-NFC expansion board for Raspberry Pi
  • SparkFun Voltage-Level Translator
  • 5 NeoPixel Diffused 8mm Through-Hole LED
  • 5 Arcade LED push buttons
  • MEAN WELL SNT RS 15 5
  • MEAN WELL SNT RS 15 12
  • NFC Tags

Assembly

There are a lot of guides in the internet how to connect buttons, leds and the nfc board to the Pi i would recommend to search for it.

I removed the default LED from the arcade buttons and used the Neopixel LED for them. The 8mm version fits perfectly into the buttons.

Operating System

The Pi is running an archlinux. I installed Python 2.7 with some extensions and the Music Player Daemon.

Code

It's a Python based project. I created a modular architecture and some of the system parameters are configurable using a config script. It should be possible to use this code with another count of LED or buttons.

Conclusion

I was pretty surprised how easy it was for my toddler to put the NFC-cards into the slot. He was able to select the next title and adjust the volume.

Acknowledgments
